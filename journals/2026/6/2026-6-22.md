# 📅 2026 年 6 月 22 日 - 日志回顾与成长记录：本地模型"发热门"与银行外包的"火速撤退"

## 🚀 核心进展与成果 (Top 5 Actions)

1.  **《历史的温度 8》：继续认识古人** 📚
    * **阅读推进**：继续阅读——每天认识几个历史人物，保持阅读节奏和饥渴感。

2.  **Codex + LM Studio：本地推理的"发热门"** 🦿
    * **野心与挫折**：想用 Claude Code 安装 Codex，配置 DeepSeek API 还不够——想用 LM Studio 本地模型替代。甚至想跑 2 个模型：Gemma-4-26b-QAT（逻辑）+ Qwen3-Coder-30b（编码）。
    * **天真了**：Codex + LM Studio 跑起来后 MacBook 发热严重。和 LM Studio 单独跑完全不同（串行 vs 并行推理）。DeepSeek 建议换小模型可减少 60% 发热——明天继续尝试。

3.  **PC 配件处置计划** 🚀
    * **行情发现**：96GB 七彩虹内存从 1599 涨到 7000 元，SSD 和内存涨价大头。整机估卖 9000-14000 元。
    * **空间优化**：腾出桌面位置，计划入 2000 元左右的高刷旋转屏接 MacBook。HDMI 直连就好，32 寸旧显示器太老了。

4.  **浦发银行外包设计：第一天就撤退** 💬
    * **需求变味**：许诺介绍了一个浦发银行"盛夏惠生活"主视觉外包。客户经理一开始正常沟通——我给了拼贴风格参考 10 多张。然后直接让我做 3 个方向的创意阐述 + PPT + 最难的 3D 效果。
    * **火速撤退**：感觉不太对——这客户经理在把我往火坑推，一周出 3 方向 3D 效果创意阐述。直接和许诺说"可能不太适合这边的工作方式和节奏"赶紧撤了。从第三方角度看时间，反复修改太不值了。

5.  **柚宝的睡眠战争：崩溃的夜晚** 🔁
    * **核心矛盾**：柚宝每晚哄睡哭闹、半夜醒要开灯、要妈妈拍睡、抢妈妈被子——老婆快被搞疯了。我暴怒抱她去卫生间打了两下屁股。
    * **根因分析**：意识到是阿婆中午拍睡养成的习惯——快 3 岁的孩子还要拍睡、半夜要开灯、要妈妈对着她睡。我们房间的规则她适应不了（关灯、自己盖被子、不拍睡）。教育方式的不统一（阿婆的宠 vs 我们的严）是根本问题。

## 🧠 知识摄取与挑战 (Learning & Challenges)

* **本地模型推理的边界**：LM Studio 串行推理不热，加 Codex 外壳变并行推理就热了。换小模型或 QAT 版本可解。
* **外包的"需求爬坡"预警信号**：第一天就做 3 方向+PPT+3D 效果——这不是合作，是甩锅。早点撤退比硬撑好。
* **遇到的挑战与应对：**
    1.  **Codex 发烫**：**对策**：换小模型或直接跑 DeepSeek API（不贵），本地推理留给简单任务。
    2.  **柚宝睡眠习惯问题**：**对策**：和阿婆统一规则——关灯睡、不拍睡、自己盖被子。但老人带娃的习惯很难改。
* **下一步计划：**
    * **明天**：继续 ComfyUI 安装 + 游戏开发尝试。
    * **下周**：关注柚宝睡眠调整效果。
